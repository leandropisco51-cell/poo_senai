O que é POO?

A **Programação Orientada a Objetos (POO)** é um modelo ou paradigma de programação.

Em vez de escrevermos códigos como uma lista de instruções sequenciais, nós organizamos o nosso sistema imitando o mundo real: através de Objetos que interagem entre si.

Por que usar?

Facilita a manutenção, o reaproveitamento de código e deixa sistemas complexos muito mais organizados.

Programação Estruturada
  • Sequência de passos
  • Funções soltas
  • Difícil de reutilizar
Orientação a Objetos
  • Módulos independentes
  • Dados + Ações juntos
  • Altamente reutilizável

Classe vs Objeto: A Grande Analogia

A Classe (O Molde)

Pense na classe como a planta de engenharia de uma casa ou a receita de um bolo. Ela não é a casa física, mas define exatamente como a casa deve ser construída.

  • Modelo
  • Planta
  • Abstrato
  • Estrutura

O Objeto (A Instância)

O objeto é a casa construída a partir da planta, ou o bolo real feito a partir da receita. Ele tem forma física, ocupa espaço e possui suas próprias características.

  • Realidade
  • Instância
  • Concreto
  • Indivíduo

A Classe: Construindo o Molde

Passe o mouse (ou toque) nas partes destacadas do código para entender sua função:

class Cachorro:
    
    def __init__(self, nome, raca):
        self.nome = nome
        self.raca = raca
        
    def latir(self):
        return f"{self.nome} Diz: Au au"

Passe o mouse sobre as linhas de código para ver a explicação passo a passo.

Declarando a Classe

A palavra-chave class diz ao Python que estamos definindo um novo molde/projeto chamado Cachorro. Por convenção, nomes de classes começam com letra maiúscula.

O Construtor (__init__)

O método __init__ é o construtor da classe. Ele roda automaticamente no momento em que criamos um novo cachorro. O parâmetro self representa o cachorro específico que está sendo criado.

Atributos (Características)

self.nome e self.raca são atributos. Eles guardam as características/dados de cada cachorro criado. Cada cachorro terá seu próprio nome e raça.

Métodos (Comportamentos)

latir(self) é um método. Métodos são funções criadas dentro da classe que representam as ações ou comportamentos que o objeto pode realizar.

Resumo dos Conceitos Chave

POO

Paradigma de programação que organiza o sistema mapeando conceitos do mundo real para códigos através de objetos.

Classe

O molde abstrato, planta ou modelo a partir do qual novos objetos são definidos e estruturados.

Objeto

A instância física, concreta e real de uma Classe. Possui seus próprios dados (atributos) e funções (métodos).
